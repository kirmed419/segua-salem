# SEGUA Mohamed Salem — Portfolio

Personal portfolio of **SEGUA Mohamed Salem** — Data Scientist & Data Analyst, Master's graduate and State Engineer in Statistics (ENSSEA, Algiers).

A hand-built, dependency-free static site: an editorial "data-lab" aesthetic with an animated data-field background, a self-drawing regression plot, and scroll-driven motion.

**Live:** https://kirmed419.github.io/segua-salem/

## Stack
- Plain HTML / CSS / vanilla JS — no framework, no build step
- Google Fonts (Fraunces, Space Grotesk, JetBrains Mono)

## Run locally
```bash
python3 -m http.server 4321
# then open http://localhost:4321
```

## Structure
- `index.html` — markup
- `style.css` — styling
- `script.js` — canvas field, regression plot, counters, scroll reveals
- `photo.png` — portrait
- `SEGUA-Mohamed-Salem-CV.pdf` — résumé
